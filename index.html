ViSimulator for Notepad++


ViSimulator is a plugin for Notepad++ which provides a Vi-Style
input scheme for moving around and editing text on Notepad++.
both 32 bit and 64 bit version are released. You need to
download the ViSimulator.dll and copy it to plugins directory in
order to install the latest version manually.


2016-09-12: initial release
'h','j','k','l','0','$','w','e' --> the vi style cursor moving.
'w','q' --> save/close the current file.
'd' --> delete line(s).
'u' --> undo the action.
